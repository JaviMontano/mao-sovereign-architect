## Descripción

**Sovereign Architect** es un plugin de arquitectura de software que actúa como Principal Software Systems Lead. Analiza código, repositorios, especificaciones, incidentes y deuda técnica para tomar decisiones arquitectónicas fundamentadas.

> **66 agentes** · **128 skills** · **120 comandos**

---

## Instalación

Agrega como plugin de Claude Code:

```json
{
  "mcpServers": {
    "sovereign-architect": {
      "command": "claude",
      "args": ["mcp", "serve", "--plugin", "mao-sovereign-architect"]
    }
  }
}
```

## Características Principales

| Característica | Descripción |
| --- | --- |
| **66 agentes de arquitectura** | Especializados en cada dominio arquitectónico |
| **Análisis de deuda técnica** | Detección, cuantificación y plan de remediación |
| **ADR** | Architecture Decision Records con trazabilidad completa |
| **C4 Model y diagramas** | Contexto, Contenedores, Componentes, Código |
| **Auditoría de seguridad y rendimiento** | Análisis profundo con recomendaciones accionables |
| **Code review multi-dimensional** | Calidad, seguridad, rendimiento, mantenibilidad |
